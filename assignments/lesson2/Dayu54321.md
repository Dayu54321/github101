## 4. Git命令使用过程总结

#创建本地仓库
- $ cd d:前往D盘
- $ mkdir git：创建一个名为git的文件夹
- $ cd git：前往git文件夹
- $ git  init：将目录变成git可管理的仓库

#克隆远程仓库到本地git文件夹
- $ git clone https://github.com/Dayu54321/github101.git

在assignments/lesson2目录下复制了example-report.md，重命名为Dayu54321.md,并用Trae来编辑。

- $ cd github101/assignments/lesson2:前往这个文件夹
- $ git add Dayu54321.md：添加文件到暂存区
- $ git commit -m"add lesson2 Dayu54321.md"：提交更改到本地仓库
- $ git status：验证是否提交完整

（以上步骤在实操过程中出了很多问题，一言难尽！花了很多时间反复调整尝试，终于顺利走到这一步!）

#本地仓库与远程仓库建联
- $ git remote add origin https://github.com/Dayu54321/github101.git：本地仓库与远程仓库地址关联

提示：error: remote origin already exists.

- $ git push -u origin master：将本地仓库内容提交到远程仓库Github上

提示：error: src refspec master does not match any
error: failed to push some refs to 'https://github.com/Dayu54321/github101.git'
